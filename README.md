PocketPlus (0.2)
================

A News+ extension for Pocket.
This project is in early stages and should be considered as beta.


Features
-------------------------------
* Authenticate with Pocket.
* Sync unread stories.
* Sync favorite (starred) storries.
* Handle read/unread requests.
* Support for tagging stories.
